# Reversi-Game

### Simple console Reversi(Othello) game implementation using NodeJS + TypeScript.

Requirements:
- `node` >= 12.8.3
- `npm` >= 6.14.6 or `yarn` >= 1.17.3

Start in dev mode:
- `npm run start:dev` or `yarn start:dev`

Start in prod mode:
- `npm run build` or `yarn build`
- `npm start` or `yarn start`
